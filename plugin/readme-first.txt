This is a template project to build plugin attributes
MyKey.cs is the plugin source
The plugin is build in net45 for using with o3pgen.exe console
and netstandard2.0 for using with the GlobalTool
 Only the plugin.dll (without referenced dll) is needed

To use, use the option
 o2pgen -r myurl --plugin source\to\plugin\folder

For more details, read wiki:
   https://github.com/moh-hassan/odata2poco/wiki/plugin
