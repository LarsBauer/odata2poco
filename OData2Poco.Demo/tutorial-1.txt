Using OData2Poco.CommandLine tool
You can run the program o2pgen from CMD command line or in Package Manager Console in Visual Studio

In Package Manager Console type:
PM>install-package OData2Poco.CommandLine
A new folder o2p  in the root folder of the project including o2pgen.exe is instaleed

cd <your project path> , example: 
PM> cd OData2Poco.Demo

PM>o2p/o2pgen
the help screen is displayed

PM> o2p/o2pgen -r http://services.odata.org/V4/Northwind/Northwind.svc -f north.cs -j -c
generate camelcase and add jsonproperty 

The file is saved north.cs including the poco source code  

you can include the file in your project
